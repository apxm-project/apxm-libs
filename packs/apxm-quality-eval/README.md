# apxm-quality-eval

Quality-eval harness operations: fixtures, budgets, judges, claim
checks. Requires the `apxm-project/apxm-eval` repo cloned as a sibling
of `apxm`.

## Install

```bash
dekk apxm libs install apxm-quality-eval
```
