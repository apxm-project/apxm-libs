# apxm-plan-as-graph

The runtime-invocable plan-as-graph dispatcher. Given a natural-language
task, emits a typed APXM AIR graph and routes it through the APXM
compiler + runtime.

## Install

```bash
dekk apxm libs install apxm-plan-as-graph
```

Or via sibling clone of `apxm-libs` next to `apxm`.
