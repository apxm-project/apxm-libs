# apxm-orient

Deterministic project orientation card. Replaces ad-hoc `ls`/`git
status`/`grep` exploration with a single structured probe. The entry
flow design is captured in `notes.md`; `skill.air` not yet compiled.

## Why this is a priority pack

It is the gateway skill: every other APXM skill assumes the agent has
context that `apxm-orient` would have surfaced. Shipping this first
makes the rest of the catalog usable without each skill re-explaining
the repo layout.
